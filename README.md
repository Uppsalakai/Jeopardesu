JEOPARDESU MANUS
==========

1. Välkommna till Jeopardesu

2. Berätta om deltagare

3. Låt deltagarna testa sina ljud ex:
Fluff trycker på sin knapp, ljud + bild spelas.

4. Gå igenom kategorier

5. Deltagarna börjar spela
